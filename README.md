# 📖 Project Overview

Social media has become an integral part of students' daily lives. While it offers opportunities for learning and communication, excessive usage can impact academic performance, mental wellbeing, and productivity.

This project transforms raw student data into an interactive Tableau dashboard that helps stakeholders identify high-risk demographic groups, compare addiction trends, and make data-driven decisions through visual analytics.

---

# 🎯 Business Problem

Educational institutions and policymakers often lack a centralized way to analyze social media addiction trends among students.

This dashboard answers questions like:

- Which age group has the highest addiction score?
- Does relationship status influence addiction?
- Which countries report higher addiction levels?
- Is addiction evenly distributed across genders?
- How does addiction vary across academic levels?

# 📊 Dashboard Features

- Compare addiction scores across different age groups.
- Identify the most vulnerable age segment.

### 🌍 Country-wise Analysis
- Analyze addiction scores across countries.
- Compare regional digital behavior.

### ❤️ Relationship Status Analysis
- Understand how relationship status influences addiction.

---

# 🔍 Key Insights

### 📈 Age Analysis
- Students aged **21** recorded the highest addiction score.
- Ages **19** and **22** followed closely.

**Business Recommendation**
- Launch digital wellbeing workshops targeting university students.

---

### 🌍 Country Analysis

Top countries by addiction score:

| Country | Score |
|---------|------:|
| India | 398 |
| USA | 344 |
| Canada | 228 |

**Business Recommendation**

Develop localized awareness campaigns based on regional behavior.

---

### ❤️ Relationship Status

- Single students showed the highest addiction scores.
- Students in relationships reported comparatively lower scores.

**Business Recommendation**

Promote offline engagement through clubs and campus activities.

---

### ⚖️ Gender Analysis

- Male: **51.1%**
- Female: **48.9%**

**Business Recommendation**

Design inclusive awareness programs rather than gender-specific interventions.
